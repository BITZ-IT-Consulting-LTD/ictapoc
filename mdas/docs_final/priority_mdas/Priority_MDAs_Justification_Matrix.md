# Priority MDAs – Justification Matrix

This document presents the prioritised list of Ministries, Departments, and Agencies (MDAs) for intervention based on the approved prioritisation model. The model assessed:

- Institutional capacity gaps (DMI and weakest pillar scores)
- Citizen/public impact and service reach
- Records volumes and transaction intensity
- Feasibility of intervention based on readiness and mandate

# 3. Prioritisation Model Summary

The prioritisation model applied three core criteria:

---

## A. Need (Capacity Gap)

Measured using:

- Digital Maturity Index (DMI)
- Weakest pillars:
  - Records Management Index (RMI)
  - Enterprise Systems Maturity Index (ESMI)
  - Governance & Compliance Index (GCI)

Priority given to:

- Foundational–Basic maturity MDAs
- Records-constrained institutions

---

## B. Impact (Citizen / Public Value)

Measured using:

- Record volumes
- Population coverage
- Sensitivity of records
- Frequency of transactions

---

## C. Feasibility (Implementation Readiness)

Measured using:

- Existing ICT systems
- Leadership readiness
- Defined registries
- Governance structures

---

# 4. Strategic Objective

The prioritised MDAs represent the highest value intervention opportunities to:

- Improve citizen service delivery
- Strengthen national registries
- Enable interoperability
- Improve accountability
- Enable digital government transformation
